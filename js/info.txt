// JavaScript events are interactions or occurrences that happen in a web browser, typically triggered by user actions or changes in the document. Here's a list of some commonly used JavaScript events:

// 1. Mouse Events:
//    - click
//    - dblclick (double click)
//    - mousedown
//    - mouseup
//    - mousemove
//    - mouseover
//    - mouseout

// 2. Keyboard Events:
//    - keydown
//    - keyup
//    - keypress

// 3. Form Events:
//    - submit
//    - reset
//    - change
//    - input (when the value of an input changes)
//    - focus
//    - blur

// 4. Document Events:
//    - load (when the document and its resources finish loading)
//    - unload (when the document is being unloaded)
//    - beforeunload (before the document is about to be unloaded)
//    - readyStateChange (when the readyState of the document changes)

// 5. Window Events:

//    - load (when the window finishes loading)
//    - unload (when the window is being closed)

//    - drop

//     - animationstart
